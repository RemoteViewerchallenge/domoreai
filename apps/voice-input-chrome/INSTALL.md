## Keyboard Shortcut

**Default:** `Ctrl+Shift+Space` (Windows/Linux) or `Cmd+Shift+Space` (Mac)

**To change it:**
1. Go to: **chrome://extensions/shortcuts**
2. Find "Voice Input"
3. Click the pencil icon
4. Press your desired keys
5. Click "OK"

## Troubleshooting

### Extension not showing?
- Make sure Developer mode is ON
- Refresh the extensions page
- Check you selected the correct folder

### Microphone not working?
- Click the microphone icon in Chrome's address bar
- Select "Always allow"
- Check system microphone settings

### Shortcut not working?
- Make sure you're in a Chrome window
- Check for conflicts at chrome://extensions/shortcuts
- Try clicking the extension icon instead
